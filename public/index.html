Quick Setup Guide

1

Install Java on Pi

sudo apt update && sudo apt install default-jdk
2

Clone Repository

git clone <repo-url> minecraft-config
3

Run Setup Script

bash scripts/setup.sh
4

Install Auto-Start

sudo bash scripts/install-systemd.sh

Common Commands

sudo systemctl start minecraft Start server
sudo systemctl stop minecraft Stop server
sudo systemctl status minecraft Check status
sudo journalctl -u minecraft -f View live logs
